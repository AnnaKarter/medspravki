АКЦИЯ !!! Медсправки выдаются бесплатно для участников специальных военных операций и членов их семей при наличии соответствующего документа и медсправок из диспансера в течении всего дня.

Для всех остальных, что бы получить медсправку бесплатно нужно:

  1. иметь при себе справки из психоневрологического и наркогологического диспансера
  2. оставить положительные отзывы о компании на сайтах: yell.ru и yandex.ru
  3. в регистратуре предъявить скриншоты или фотографии отзывов
    4.

После этого - бесплатно проходите медосмотр и бесплатно получаете медицинскую справку!

Милости просим на Час Удачи!

Предварительная запись по телефону 8 499 235 77 00

Бесплатный осмотр проходит каждое воскресенье, утром, с 9:30 до 10:30 часов.

Наша медкомиссия предоставляет следующие услуги:

  • справка для бассейна
  • справка для спортсекции
  • освидетельствование и оформление медицинских справок:для ГИБДД
  • освидетельствование врачами психиатром и психиатром-наркологом
  • на оружие, для работы в охране (ЧОП) по форме № 002-О/у
  • на работу или учёбу по форме № 086/у (выдается до 20:00)
  • на управление маломерными судами
  • в дома отдыха и пансионаты
  • на госслужбу по форме 001/ГСу
  • на гостайну по форме 989
  • на должность судьи по форме 086-1/у
  • справка по приказу 302Н (для работы)
  • справка для студентов по форме 095/у
  • справка на работу по форме 282
  • справка для выезжающих за границу 082/у
  • санаторно-курортная по форме 072/у
  • справка для получения путевки по форме 070/у

* выдача медицинских справок производится после прохождения врачебного осмотра

- заключение психолога для ГИБДД

- ЭЭГ обследование

- выдача больничных листов

  • Больничный лист - Первичный прием врача терапевта - 1200 рублей
  • Больничный лист - Вторичный прием врача терапевта - 900 рублей
  • Больничный лист с выездом врача терапевта на дом (в пределах МКАД) первичный - 2000 рублей
  • Больничный лист с выездом врача терапевтана дом (в пределах МКАД) вторичный - 2000 рублей
  • Прием медицинской сестры на дому (в пределах МКАД) без стоимости манипуляций - 1000 рублей

- оформление медицинских книжек

- ЭКГ

- прием психиатра/наркологa

- консультации специалистов

Для оформления справки при себе нужно иметь: паспорт или иной документ удостоверяющий личность

Для переоформления прав при себе нужно иметь водительское удостоверение

Порядок оплаты: в отношении физических лиц - через кассу, в отношении юридических лиц - в безналичном порядке

Все врачи и иные медицинские работники являются дипломированными специалистами
